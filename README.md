# 2026 NBA Championship Analytics | Python + Power BI

A complete end-to-end sports analytics project that analyzes the 2026 NBA Playoffs and Finals using Python and Power BI.

The goal of this project is to build a professional data analytics pipeline—from collecting raw NBA data to creating an interactive dashboard that uncovers player, team, and series insights.

This project demonstrates skills in:

- Data Collection
- Data Cleaning
- Data Engineering
- Exploratory Data Analysis
- Data Modeling
- DAX
- Power BI
- Sports Analytics

## Project Goals

- Collect NBA Playoff and Finals data using Python
- Build a clean analytics dataset
- Perform exploratory analysis
- Create advanced Power BI dashboards
- Generate basketball insights using statistics
- Practice professional data engineering workflows
